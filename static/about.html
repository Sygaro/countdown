Hva er dette?

En enkel, robust nedteller for Raspberry Pi-kiosk. Støtter daglig tidspunkt, engangs-tidspunkt, varighet og stopp-skjerm (tekst/bilde/blackout).

Appen kjører Flask (Gunicorn) og renderes i Cog/WPE WebKit. Konfig lagres i config.json med atomiske skriv.

API (utdrag)

  • GET /api/config → {config, tick}
  • POST /api/config → lagre + bytte modus (se admin)
  • POST /api/start → start varighet {"minutes": N}
  • POST /api/stop → stopp varighet (→ daily)
  • GET /tick → status (inkl. cfg_rev)
  • GET /health → 200 ok
